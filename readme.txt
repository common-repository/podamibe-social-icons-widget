== Description ==

<strong>Podamibe Social Widget</strong> is a <strong>Free WordPress plugin</strong> to allow anyone easily show their social sites icons with respective links by the use of the user friendly and easier widget settings!

== Installation ==

1. Upload the plugin 'podamibe-social' to the `/wp-content/plugin s/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Appearance->Widgets->Podamibe Social Widget
4. Drag the widget to your desired sidebar and fill the fields and save


== Frequently Asked Questions ==

= What is main title? =
This is the title to be displayed over the widget title.

= What is content? =
This is field to insert the content or short description below the title.

= What are the color options? =
Icon color -> Color for social icons
Icon hover color -> Hover color of icons
Background color -> Icons background color
Background hover color -> Icons background hover color 

= What are the desired theme options? =
Select the icons style which you want to display on front

= What is social sites list? =
This is where the icons and their respective links appears

= How to insert icons and links? =
At first after inserting widget click save button than click Add New button than there appears the field to insert icons and links than insert the respective icons with thier links.

= Which icon to insert? =
You can insert icons from fontawesome and only icon name should be inserted. For example to insert facebook icon you have to insert only 'facebook'.
